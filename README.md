Life Savior: Healthcare & Emergency Directory
Life Savior is a web-based healthcare platform designed to bridge the gap between patients and medical professionals in Dhaka City. It provides a centralized directory for various medical specialists and a comprehensive list of emergency contact numbers for critical situations.
